前言——站台零点

发表于 2021-08-09 分类于 随笔 Disqus：
本文字数： 723 阅读时长 ≈ 1 分钟

这篇文章大概会是一系列稀奇古怪的文章的开始。

0

在去年圣诞节前的一个冬夜，我曾经对着新买的日记本发呆，憧憬着，下定决心要让自己的生活变得有些节奏，有些乐趣。从那时起，我大概一不小心接触尝试了一大堆新东西——其中的乐趣倒是不少，可惜缺了一些章法。而且，随着日子一天天变得更忙，周围的信息一天天变得更多，面临的任务一天天变得更怪异多变，我想我大概更需要一个清醒的脑子，让活蹦乱跳的神经元至少稍微排排队。建立这个驿站的理由大抵如此。

1

那，怎么样才能让这堆神经元稍微整齐一点呢？这大体来说会涉及到几个方面：

  • 首要任务自然是把跳动的神经拎出来。那么，这个驿站大概会记录一些有趣的，值得铭记的经历，包括事情本身，它们在我的脑海里留下的印象，以及与之相关的事物和人。
  • 其次，这些不同的经历、事物和人大抵有着相同、相似或不同的规律。找到这类相似之处的内容大概也会在这个驿站里面做客。
  • 但是这堆规律不见得会长成我乐意看到的样子。驿站需要管理，而我是驿站这个小世界的主人。我或许会写一写我希望的小世界。
  • 那么，既然有了事实作为前奏、规则作为展开，有了期望的目标，驿站的交响曲大概就可以迎来它的最终乐章了。如果一切顺利，我大概会记录下我要如何实现梦中的世界。

既然这样，那我们就开始每周末在驿站冒泡吧。

一些无关紧要的话

本网站的内容不针对任何人。所有出现的名字均属于其他鳄鱼。

本网站出现的任何新闻事件与真实世界无关。所有新闻事件均发生于鳄鱼驿站背后的沼泽地。

本网站的内容与当今军事与政治无关。沼泽地里打架没那么热闹。

如无准许，请勿传播本网站的内容——这只鳄鱼喜欢安静。

本网站的内容可能会出现在其他平台上，尤其是那些有趣的，值得铭记的经历，以及我希望的小世界。
